# Gameonix  WebPage 🎮

This project is a frontend clone of the Gameonix website created using **HTML and CSS**.  
It was built for practicing **frontend development, responsive design, and UI layout structuring**.

---

## 🚀 Features

- Responsive homepage layout
- Modern gaming-style UI
- Navigation bar
- Game cards / sections
- Footer section
- Clean and structured CSS
- Animations
---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

gameonix-clone

├── index.html

## 🎯 Purpose of the Project

This project was built to:

- Practice **HTML and CSS layout design**
- Improve **frontend development skills**
- Learn how real gaming websites structure their UI
